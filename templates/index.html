What is Snarky Squirrel?

An open-source multi-agent AI system that reviews GitHub pull requests for code quality, security vulnerabilities, and best practices — powered by LangGraph and Claude.

  1. 1 Paste a GitHub PR URL (e.g. https://github.com/owner/repo/pull/42) in the input below.
  2. 2 Click Review PR — four specialized agents run in parallel: Security, Code Quality, PR Review, and Summary.
  3. 3 Receive a scored report (0 – 10) with severity-ranked findings and a plain-English summary.
  4. 4 Optionally post the review as a comment directly to GitHub via the checkbox below.
